MongoDB

1.1. Guía instalación de MongoDB:http://docs.mongodb.org/manual/tutorial/install-mongodb-on-linux/1.2. Para entrar modo consola se escribe el comando:mongo1.3. Para mostrar las base de datos:show dbs1.4. Para usar una base de datos (sino existe la crea) escribimos:use "nombre_bd"1.5. Para mostrar las coleciones (tablas):show collections1.6. Hay dos formas de crear colecciones:a) db.ejemplo.insert( {"ejemplo": "mongodb"} )
b) a = { "ejemplo":"mongodb", "capacitacion":"mongodb", "fecha":"2014-01-03" }
db.ejemplo.insert( a )1.7. Para borrar una colección:db.ejemplo.drop()1.8. Para obtener información de las tablas:db.ejemplo.find()     -- regresa todos los registros  (select * from con SQL) db.ejemplo.find().count() -- regresa el número de registros (select count(*) from)

Segunda Parte

1.9. Para obtener todos los registros de una fecha:db.ejemplo.find({fec_registro:"2014-01-03"} ) -- regresa un json con los documentos encontrados.1.10 Para obtener todos los registros de una fecha en orden:db.ejemplo.find({fec_registro:"2014-01-03"} ).sort(id:-1) -- ordena descendente en base al id1.11 Para obtener un sólo registro:db.ejemplo.findOne({ fec_registro:"2014-01-03"})1.12 Para borrar registrosdb.ejemplo.remove( {fec_registro:"2014-01-03"}) -- delete from tabla where fec_registro = "2014-01-03"1.13. Para borrar todos registros de una coleción:db.ejemplo.remove()     -- equivalente delete from table1.14. Para actualizar registrosdb.ejemplo.update({ fec_registro:"2014-01-03" }, { $set:{"capacitacion":"mongodb2"}} )
